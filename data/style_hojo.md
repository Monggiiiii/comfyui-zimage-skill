# Hojo Tsukasa Style Prompt Guide

## Trigger Word
```
Zanshou_kin_Hojo
```

## Core Features
- **Era**: 80s City Hunter style
- **Color**: Pure black and white manga (must use monochrome, greyscale)
- **Texture**: Screentone, ink lines, light and shadow
- **Theme**: Urban, sexy, mature women

## Prompt Structure Template
```
[trigger word], [shot type], [angle/composition].
[pose (professional terminology)].
[clothing + material + texture + wearing state].
[anatomical details: bones/muscles/curves].
[expression/demeanor].
[background/environment].
Lighting: [light source direction] + [lighting effect] + [interaction with body].
[rendering technique], [atmosphere], monochrome.
```

## Required Tags
| Type | Tags |
|------|------|
| Shot Type | artistic nude, anatomy study, masterpiece, intimate shot |
| Style Limit | monochrome, greyscale, manga style, 80s retro |
| Subject Limit | 1girl, solo |

## Pose Terminology
| Term | Meaning |
|------|---------|
| contrapposto | Classical balanced standing pose |
| S-curve pose | S-shaped curve posture |
| odalisque pose | Classical reclining pose |
| over the shoulder look | Looking back pose |
| back arched | Arched back |

## Material Textures
| Material | Texture Description |
|----------|---------------------|
| satin | liquid shine, lustrous |
| velvet | rich texture, deep shadows |
| silk | flowing, pooling at elbows |
| leather | shiny, glossy, glistening |
| lace | intricate, delicate, see-through |

## Lighting Description
```
Lighting: [light source type] from [direction], [lighting effect], [interaction with body].

Examples:
- soft studio lighting from the side to accentuate muscle volume
- dramatic rim lighting highlighting the silhouette and curves
- cool blue-tinted backlight from window, strong rim light on figure edge
```

## Rendering Techniques
- screentone shading/gradient/texture
- halftone rain texture
- cross-hatching on leather/velvet
- chiaroscuro
- clean/delicate ink lines
