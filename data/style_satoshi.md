# Satoshi Urushihara Style Prompt Guide

## Trigger Word
```
sato
```

## Core Features
- **Era**: 90s/early 2000s Japanese anime, OVA movie quality
- **Texture Core**: High-gloss skin (shiny skin), reflective clothing, sweat/wet effects
- **Theme Preference**: Sci-fi cyberpunk, fantasy female warriors, space opera
- **Lighting Style**: Strong rim light, neon colors, high contrast
- **Reference Works**: Ghost in the Shell, Record of Lodoss War, Patlabor, Tekkaman Blade

## Prompt Structure Template

### Template A: Tag Stacking Style
```
[trigger word], ([style definition]:weight), [character basics], [texture description], [clothing/equipment], [pose/action],
([lighting/environment]:weight), [background], [technical effects], [atmosphere words].
```

### Template B: Cinematic Description Style
```
[trigger word], [shot type description]. [Detailed scene and character description]. [Clothing material details]. [Lighting and atmosphere description]. [Technical style words].
```

```
90s anime style, retro anime poster, cel shading
1990s (style), retro anime aesthetic, retro artstyle

## Theme Categories

### A. Sci-Fi/Cyberpunk
```
Clothing: plugsuit, tactical suit, neural interface suit, form-fitting, glossy
Mechanical: cyborg, mechanical parts, data cables, wires, armor, pauldrons
Environment: city lights, neon lighting, holographic, industrial, rain, night
```

### B. Fantasy Warrior
```
Equipment: armor, breastplate, pauldrons, holding sword, weapon, shield
Elements: elf, pointy ears, angel wings, magic fire, pyrokinesis
Environment: fantasy landscape, mountains, sunset, golden hour, medieval
```

### C. Space/Universe
```
Clothing: plugsuit, space suit, leotard, highleg leotard
Environment: floating in space, galaxy background, starry sky, nebula
Effects: floating hair, weightless, zero gravity, backlighting
```

## Lighting Description

| Type | Tags |
|------|------|
| Rim light | rim light, strong rim light, backlighting |
| Neon light | neon lighting, pink and cyan lights |
| Golden hour | golden hour lighting, warm orange light, sun flare |
| Chiaroscuro | chiaroscuro, deep shadows, film noir |

### Lighting Combination Examples
```
Sci-fi rainy night: backlighting, strong rim light, neon lighting, pink and cyan lights
Sunset fantasy: sunset, intense golden hour lighting, warm orange light, sun flare
Cyberpunk: chiaroscuro lighting, deep shadows, neon glow, cold cyan tones
```

```

Inside the dark, fluid-filled cryo-chamber, a girl with floating silver-white hair is suspended in bioluminescent cyan liquid. Lighting: soft internal glow illuminating her shiny skin and suit, contrasting with pitch black industrial lab outside. Water droplets slide down outer glass surface. Strong refraction, chromatic aberration, sharp cel shading, film grain.
